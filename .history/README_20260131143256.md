## AI/ML Portfolio

Production-grade portfolio built with Next.js 14, TypeScript, Tailwind CSS, Framer Motion, GSAP, and React Three Fiber.

### Setup

1. Install dependencies:
	- npm install
2. Create a local environment file:
	- Copy .env.example to .env.local and fill in values.

### Development

- npm run dev

### Build

- npm run build

### Environment Variables

- RESEND_API_KEY: API key for Resend.
- CONTACT_TO_EMAIL: Recipient email for contact form.
- GITHUB_USERNAME: GitHub username for live stats.
- GITHUB_TOKEN: Optional token to avoid rate limits.
